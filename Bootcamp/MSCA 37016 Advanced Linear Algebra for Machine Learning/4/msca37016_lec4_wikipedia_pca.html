Step 2: Extract Wikipedia articles¶

https://pypi.org/project/Wikipedia-API/

In [2]:
import wikipediaapi

wiki = wikipediaapi.Wikipedia('en')
corpus = {s: wiki.page(s).text for s in sovereign_states}

Step 3: Transform articles into text document vectors¶

https://scikit-learn.org/stable/modules/generated/sklearn.feature_extraction.text.CountVectorizer.html

In [3]:
from sklearn.feature_extraction.text import CountVectorizer

model = CountVectorizer(strip_accents='unicode',
                        lowercase=True,
                        token_pattern='[a-z]+',
                        stop_words='english')

x = model.fit_transform(corpus.values()).toarray()
In [4]:
x.shape # (num document)x(num token)
Out[4]:
(206, 70734)

Step 4: Normalize text document vectors¶

In [5]:
import numpy as np

x = x / np.expand_dims(np.linalg.norm(x, axis=1), 1)

Step 5: Perform PCA on text document vectors¶

https://scikit-learn.org/stable/modules/generated/sklearn.decomposition.PCA.html

In [6]:
import sklearn.decomposition

model = sklearn.decomposition.PCA().fit(x)

Step 6: Visualize the scree plot of % variance explained¶

https://en.wikipedia.org/wiki/Scree_plot

In [7]:
%matplotlib inline
%config InlineBackend.figure_format = 'retina'

import matplotlib.pyplot as plt

plt.figure(figsize=(12, 6))
plt.plot(model.explained_variance_ratio_.cumsum() * 100, '.')
plt.grid(linestyle=':')
plt.gca().set_axisbelow(True)
plt.title(f'PCA of Wikipedia Articles of {num_states} Sovereign States')
plt.xlabel('Number of Principal Components')
plt.ylabel('Percent of Variance Explained (in %)')
plt.show()
